# Assets Folder

Esta pasta contém todos os assets do projeto Unity.

## Estrutura

- **Animations/** - Arquivos de animação (clips, controllers, avatars)
- **Audio/** - Música e efeitos sonoros
- **Materials/** - Materiais e texturas
- **Models/** - Modelos 3D (FBX, OBJ, etc.)
- **Prefabs/** - Prefabs reutilizáveis
- **Scenes/** - Cenas do jogo (MainMenu, GamePlay, etc.)
- **Scripts/** - Código C#
  - Core/ - GameManager, SceneController
  - Gameplay/ - Lógica de jogo
  - Network/ - Photon/multiplayer
  - UI/ - Scripts de interface
  - Animation/ - Controllers de animação
  - Audio/ - AudioManager
- **UI/** - Sprites, ícones, fontes para interface
- **Resources/** - Assets carregados dinamicamente em runtime
